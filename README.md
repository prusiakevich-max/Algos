Практическая работа: Решение судоку с помощью backtracking

---

1. Название задания

Реализация алгоритма решения судоку методом backtracking (возврата)

---

2. Описание работы алгоритма с синтаксисом (на Python)

Алгоритм использует стратегию "пробуй и откатывайся" для заполнения пустых клеток судоку. Он основан на следующих наблюдениях:

· Если все клетки заполнены — судоку решено.
· Если клетка пустая — пробуем поставить цифру от 1 до 9.
· Цифра должна быть уникальной в строке, столбце и блоке 3×3.
· Если цифра подходит — рекурсивно решаем оставшуюся часть.
· Если решение не найдено — откатываемся и пробуем следующую цифру.

Используемые операторы и конструкции:

· for — для перебора строк, столбцов и цифр.
· if — для проверки условий (пустая клетка, валидность цифры).
· %, // — для определения границ блоков 3×3.
· Рекурсивный вызов функции solve_sudoku() — для решения подзадач.
· Возврат False и сброс значения — для backtracking.

Как работает:
Находим первую пустую клетку. Для цифр от 1 до 9 проверяем валидность в текущей позиции. Если цифра валидна, ставим её и рекурсивно решаем остаток. Если рекурсия возвращает True — решение найдено. Иначе откатываемся (backtrack) и пробуем следующую цифру.

Код на Python:
def solve_sudoku(board):
    def is_valid(row, col, num):
        # Проверка строки
        for i in range(9):
            if board[row][i] == num:
                return False
        
        # Проверка столбца
        for i in range(9):
            if board[i][col] == num:
                return False
        
        # Проверка блока 3x3
        start_row, start_col = 3 * (row // 3), 3 * (col // 3)
        for i in range(3):
            for j in range(3):
                if board[start_row + i][start_col + j] == num:
                    return False
        return True
    
    for row in range(9):
        for col in range(9):
            if board[row][col] == 0:
                for num in range(1, 10):
                    if is_valid(row, col, num):
                        board[row][col] = num
                        if solve_sudoku(board):
                            return True
                        board[row][col] = 0  # backtrack
                return False
    return True

---

3. Временная сложность

O(9^(n×n)), где n = 9 для стандартного судоку

---

4. Объяснение временной сложности

В худшем случае алгоритм перебирает все возможные комбинации цифр для каждой пустой клетки. Для сетки 9×9 с m пустыми клетками верхняя оценка — O(9^m). Backtracking значительно сокращает перебор за счёт раннего отсечения невалидных ветвей, когда цифра нарушает правила судоку. Однако формальная сложность остаётся экспоненциальной.

6. Ответ на контрольный вопрос

Сколько существует перестановок множества из n элементов?

Ответ: n! (факториал n)

Факториал вычисляется как произведение всех целых чисел от 1 до n:

· n = 3: 3! = 3 × 2 × 1 = 6 перестановок
· n = 5: 5! = 5 × 4 × 3 × 2 × 1 = 120 перестановок
· n = 10: 10! = 3 628 800 перестановок
